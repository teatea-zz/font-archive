**디자인 영감을 위한 퍼스널 폰트 라이브러리**

이 프로젝트는 디자이너 개인이 사용하기 편한 폰트의 사례를 기록하고, 수집하는 용도로 제작되었습니다. 폰트의 메타데이터와 예시 이미지를 기록하고 관리하는 개인용 아카이브입니다.

Tech Stack: Next.js, Supabase, Tailwind CSS.

**버전**: v1.6 (Phase 6 완료)  
**마지막 업데이트**: 2026-02-13  
**Phase 6 업데이트**: 폰트 관리 기능(상세보기, 편집, 삭제), 즐겨찾기 토글, 데이터 연동(Binding) 오류 해결 및 스토리지 클린업 자동화
